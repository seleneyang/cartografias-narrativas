cómo se conecta todo

El JSON exportado ahora incluye lat y lng. Si sumás los JSON de varias personas en cartas.json, la galería dibuja cada tarjeta con su mini-mapa y además un mapa general abajo con todos los puntos juntos.

El geocoding usa Nominatim (OpenStreetMap), gratuito y sin API key. Si llega a fallar, podés arrastrar el pin a mano.
